First thought

What permissions do i have?
- `ls -ld` 
  - use `-d` to see info about the current folder (not its list) 
- `level00 level00` 
  - I am level00. I am the owner. I am the group.
- `dr-xr-x---+` 
  - owner (me) and group members (my group) can `read/exec` (not write) 
```b
level00@SnowCrash:~$ ls -l
total 0
level00@SnowCrash:~$ ls -ld
dr-xr-x---+ 1 level00 level00 100 Mar  5  2016 .
```

Let's check if there are more interesting files

```b
> find / -user level00 
> find / -user level00 2>/dev/null
> find / -user flag00 2>/dev/null
> find / -user flag00 -exec cat {} \; 2>/dev/null
OR
> find / -user flag00 -exec cat {} + 2>/dev/null
cdiiddwpgswtgt
cdiiddwpgswtgt
```

- `2>/dev/null` : redir stderr output to `/dev/null`
- `\;` : mark the end of `-exec` and runs `cat` on one file a time
- `/` : from root directory
- `+` : forces `find` to run `cat` on many files at once

a = ord('a')
