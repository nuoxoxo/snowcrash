- command explained
    - `/` - from root directory
    - `+` - forces `find` to run `cat` on many files at once and not one at a time
    - `2>/dev/null` - redir stderr output to `/dev/null`
- decipher ___`cdiiddwpgswtgt`___ in python

```py
a = ord('a')
```
